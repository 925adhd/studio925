# Studio 925 Website

This repository contains the production source code for studio925.design.

## Tech Stack
- React
- TypeScript
- GitHub Pages (static deployment)
- Cloudflare DNS

## Deployment

To deploy updates:

1. Run: npm run build
2. Commit updated dist folder
3. Push to main branch
4. GitHub Pages publishes automatically
